Welcome to RequireJS

How to implement RequireJS to a project and how to load JS libraries like JQuery and plugins like JqueryTools with RequireJS

How To Implement

Add to your include headinit

<script data-main="js/main.js" src="js/libs/requirejs/require.js"></script>

In main.js set the configs

require.config({
  //By default load any module IDs from js/
  baseUrl: 'js',
  //define key lib locations
  paths: {
    jquery: "libs/jquery/jquery-1.8.1.min",
    jquerytools: "libs/plugins/jquery.tools.min"
  },
  shim: {
    'jquerytools': ['jquery']
  }
});

To require a module

 require (['modules/text'],function(){
 });

To define a module

define(['jquery'], function($) {
    $(".hello").click(function(event) {
        $(this).append("<strong>Hello World</strong>");
    });
});
